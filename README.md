# GitHub Skyline CLI extension

## Installing

```bash
gh extension install github/gh-skyline
```

## Upgrading

```bash
gh extension upgrade skyline
```

## Usage

Get the Skyline data for the current user and year:

```bash
gh skyline
```

Get the full Skyline data for the current user:
```bash
gh skyline --full
```

```bash
gh skyline â€“-user larskristianhaga
```

Get the Skyline data for specific years:

```bash
gh skyline --year 2015-2025
```
